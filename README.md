## installation

This isn't on `crates.io` as it's already similar to [iroh's quickstart
`transfer`](https://www.iroh.computer/docs/quickstart) example.

```sh
cargo add iroh-cli --git "https://github.com/1git2clone/iroh-cli/"
```
